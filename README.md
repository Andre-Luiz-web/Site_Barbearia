📌 Descrição do Projeto
Este é um projeto pessoal desenvolvido com o objetivo de criar um site para agendamento de horários em uma barbearia.
O foco principal foi oferecer uma interface moderna, responsiva e intuitiva, permitindo que os clientes escolham seu barbeiro preferido, definam data e horário, e adicionem observações importantes para o atendimento.

🛠️ Tecnologias Utilizadas
HTML5 → Estruturação das páginas e formulários.

CSS3 → Estilização com design responsivo, cores personalizadas e layout moderno.

JavaScript (ES6) → Validações de formulário (datas, horários, telefone), interatividade e mensagens de confirmação.

Google Fonts → Tipografia personalizada para dar identidade visual ao projeto.

⚙️ Funcionalidades
Agendamento online com formulário centralizado e estilizado.

Seleção de barbeiro preferido, incluindo opção de atendimento por qualquer profissional.

Validação de dados:

Bloqueio de datas passadas.

Restrição de horários fora do funcionamento da barbearia (08h às 21h).

Verificação de telefone com mínimo de 9 dígitos.

Confirmação de agendamento com feedback imediato para o usuário.

Design responsivo que se adapta a diferentes tamanhos de tela.
